# けふタイマー (Kefu Timer)

## アプリの目的

このアプリは、チームでの共同作業をより生産的にするためのデスクトップタイマーです。時間管理とBGM再生を組み合わせることで、参加者全員の集中力維持をサポートします。

既存のタイマーアプリでは実現しにくい、**「カウントダウン中にBGMが途切れることなく流れ続ける」** という機能に特化しています。

---

## 主な機能

* **シンプルなカウントダウンタイマー**:
  * 1分単位、1秒単位での自由な時間設定。
  * 1分～10分まですぐに設定できるプリセットボタン。
* **時刻超過モード**:
  * タイマーが0になった後も停止せず、超過時間をカウントアップして表示します。
  * 最大99分59秒まで計測可能です。
  * 超過中はタイマーが赤色になり、一目で時間が過ぎていることがわかります。
* **途切れないBGM再生機能**:
  * タイマー実行中、BGMがループ再生され続けます。
  * BGMはアプリ内のリストからいつでも変更可能です。
* **豊富なカスタマイズ性**:
  * BGMとタイマー終了音を、ご自身の好きな音楽ファイル（mp3, wav, ogg）に変更できます。
  * BGMのミュート機能。
* **柔軟なウィンドウ表示**:
  * 「常に最前面に表示」機能で、他の作業をしながらでもタイマーを確認できます。
  * ウィンドウサイズを自由に変更でき、タイマーの数字も大きさに合わせて見やすく変化します。
* **タスクバー連携**:
  * タイマー実行中、タスクバーのアイコンに進捗が表示されるため、ウィンドウが隠れていても残り時間を確認できます。

### 実行に必要な環境

このアプリケーションを動作させるには、**.NET 8.0 Desktop Runtime** が必要です。

お手数ですが、以下のリンクからインストーラーをダウンロードし、お使いのPCにインストールしてください。

**[.NET 8.0 Desktop Runtime (Windows x64) のダウンロード](https://dotnet.microsoft.com/ja-jp/download/dotnet/8.0/runtime?runtime=desktop&os=windows&arch=x64)**

上記リンクから「x64」版のインストーラーをダウンロードして実行すれば、アプリケーションが起動できるようになります。
