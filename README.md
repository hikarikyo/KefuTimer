# KefuTimer

## アプリの目的

このアプリは、リモートワークやチームでの共同作業をより生産的にするためのデスクトップタイマーです。Miroのタイマー機能のように、時間管理とBGM再生を組み合わせることで、参加者全員の集中力維持をサポートします。

既存のタイマーアプリでは実現しにくい、**「カウントダウン中にBGMが途切れることなく流れ続ける」**という機能に特化しています。

---

## 主な機能

* **シンプルで分かりやすいUI**: 作業の邪魔にならない、視認性の高いシンプルなデザインを目指します。
* **カウントダウンタイマー**: 自由に時間を設定し、カウントダウンを実行します。
* **BGM再生機能**: タイマーと連動し、カウントダウン中に指定したBGMを再生し続けます。
* **カスタム可能なBGM**: アプリ本体とは別に用意した音楽ファイルを読み込み、BGMとして使用できます。

---

## 今後の作業ロードマップ

このプロジェクトは、以下のフェーズで段階的に開発を進めます。

### フェーズ1: タイマー機能の実装

まず、アプリの核となるタイマー機能を完成させます。

1.  **UIの作成**: タイマーの数字、開始/一時停止ボタン、リセットボタンを配置します。
2.  **カウントダウンロジック**: ユーザーが設定した時間から正確にカウントダウンするロジックを実装します。
3.  **動作確認**: タイマーが正しく動作することを確認します。

### フェーズ2: BGM再生機能の実装

次に、タイマーと連動する音楽再生機能を追加します。

1.  **BGM再生ロジック**: 音楽ファイル（`*.mp3`など）を読み込み、再生・停止できる機能を実装します。
2.  **タイマーとの連携**: タイマーの開始と同時にBGMを再生し、タイマーが終了したらBGMも停止するようにロジックを組み込みます。
3.  **BGMのファイル選択**: アプリの実行ファイルとは別のフォルダにある音楽ファイルを読み込む仕組みを作ります。

### フェーズ3: リファクタリングとパッケージ化

最後に、コードを整理し、他の人が使えるようにアプリをパッケージ化します。

1.  **コードの整理**: 全体のコードを見直し、分かりやすく保守しやすい形に整えます。
2.  **配布用ビルド**: アプリをビルドし、実行ファイル（`.exe`）とその周辺ファイルをまとめます。
3.  **ドキュメント作成**: ユーザーがアプリを簡単に使えるよう、簡単な説明書を作成します。
